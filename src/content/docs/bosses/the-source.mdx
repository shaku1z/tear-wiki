## Phases

4 phases total. Phase transitions are triggered by HP thresholds.

### Phase 1 — The Cycle

Cycles through mechanics from all four previous bosses every 2.5s — shockwaves, sweepers, cross-fire, copy attacks. Each mechanic is familiar but the pacing is relentless.

:::tip[Strategy]
You've seen all of these before. Treat each cycle cast as a test of everything you've learned.
:::

### Phase 2 — Floor Collapse (≤62% HP)

Tears platforms out of the arena, one every 1.3s. Eventually only the floor remains. No safe platforms to land on — you must stay mobile.

:::tip[Strategy]
The floor stays. Aerial Rave users thrive here — stay airborne as long as possible.
:::

### Phase 3 — Fake Death & Regen (≤34% HP)

Collapses and regenerates at 5%/s back to 46% HP. Same as Aldric's fake. Keep the pressure up with throws — don't stop attacking.

:::tip[Strategy]
You know this trick — you've seen Aldric do it. Slam it while it regenerates.
:::

### Phase 4 — True Form (after regen)

Returns at 46% HP with all mechanics active simultaneously. Full aggression. The final test.

:::tip[Strategy]
Use every ability you have. This is what the whole run was building toward.
:::
