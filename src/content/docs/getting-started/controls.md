## Blade
| Input | Action |
|---|---|
| Hold left-click | Tether and aim the blade |
| Release left-click | Swing |
| Right-click | Throw or recall the blade |

## Movement
| Input | Action |
|---|---|
| A / D | Move |
| W / Space | Jump |
| S | Crouch or drop through one-way platforms |
| Shift / double-tap | Dash |

The blade uses spring physics: aim establishes tension, and a committed release turns that tension into a swing. See [The Blade](/mechanics/the-blade), [Dash](/mechanics/dash), and [Combat](/mechanics/combat) for live mechanical parameters.

## Touch

Mobile uses a movement joystick, a blade gesture area, and contextual dash/recall controls. Aim with a hold, release to swing, and swipe to throw.
