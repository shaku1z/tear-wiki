## Game

### v0.1 — Initial Release

*Game version: v0.1*

- Spring-physics blade system with full melee, throw, and recall mechanics.
- 5 biomes (Ruins, Canopy, Depths, Sanctum, Apex).
- Adventure, Endless, Gauntlet, Playground, Tutorial, Boss Test, and Enemy Test modes.
- Full style meter system (NICE → TEARING!).
- 6 enemy types (Charger, Ranged, Flyer, Bomber, Armored) + support types.
- 6 affixes (Tank, Swift, Rapid, Volley, Armed, Warded) and Elite escalation system.
- 3 preset enemy sub-types (Gunner, Brute, Sentinel).
- 3 status effects (BLEED, BURN, MARK).
- Achievement system (22+ achievements across 5 categories).
- Daily challenge system (deterministic, date-seeded, 10 challenge pool).
- Meta shop with Shard currency.
- Touch control support for mobile.
- Pointer lock mode for PC precision aiming.
